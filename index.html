¿Quiénes Somos?

Somos un equipo comprometido con la innovación tecnológica aplicada a la agricultura urbana sostenible. Nuestra misión es integrar soluciones automatizadas y eficientes para mejorar el cultivo en espacios reducidos, promoviendo el uso de energías renovables y tecnologías inteligentes.

Objetivo

Implementar un invernadero automatizado indoor, integrando tecnologías de control y eficiencia energética para el cultivo de ají en zonas urbanas.

Resultados

  • Se logró implementar un sistema de control ambiental completamente funcional, capaz de mantener estables las variables críticas como temperatura, humedad del suelo y nivel de agua mediante sensores, actuadores y programación adaptativa.
  • La Raspberry Pi 4 centraliza el control, almacenamiento y comunicación del sistema, permitiendo un monitoreo remoto en tiempo real a través de una interfaz web intuitiva.
  • Se implementó un sistema de iluminación con control horario que simula los ciclos de día y noche, optimizando el fotoperiodo de la planta.
  • El sistema energético fotovoltaico, conforme a la Resolución 1715 de 2014, garantiza el abastecimiento eléctrico del invernadero, alineado con criterios de sostenibilidad y eficiencia.
  • La recolección de datos históricos permitió implementar mantenimiento predictivo, mejorando la disponibilidad y reduciendo el tiempo de inactividad.
  • Se aplicaron herramientas de mejora continua como las 5S, el sistema Kanban y el enfoque Kaizen, facilitando la organización del mantenimiento y fomentando una cultura de calidad.

Elementos de Control

Actuadores controlados por relés:

  • 2 bombas de agua: una para riego y otra para evacuación del agua residual.
  • 2 ventiladores: ubicados en el lateral izquierdo, para disminuir la temperatura y ventilación.
  • Reflector LED: colocado en la parte superior para aumentar la temperatura y brindar luz al ají.
  • Bombillo LED (rojas y azules): para estimular el crecimiento del ají.

Elementos de comunicación:

  • Sensores:
    • Sensor de temperatura DS18B20 (digital): Mide temperatura del ambiente dentro del invernadero.
    • Sensor de humedad del suelo (analógico): Mide la humedad del suelo donde se ubica el ají.
    • Sensores de nivel de agua (analógicos): En el tanque principal y la bandeja de agua residual.
  • Cámara (Picamera2): Librería de Python que permite capturar la imagen en vivo al interior del invernadero.
  • Raspberry PI 4 Model B: Es la unidad de control y procesamiento del invernadero. Ejecuta un servidor Flask que permite recoger los datos de los sensores, controlar los relés y mostrar la interfaz web.
  • Serveo (SSH túnel): Genera un enlace público para acceder a la interfaz web remotamente.

Conclusiones

El modelo propuesto no solo es replicable, sino también escalable, representando una solución viable para el cultivo en espacios reducidos, con autonomía energética y gestión remota de variables críticas para el crecimiento vegetal.

El uso de estándares internacionales como las normas IEEE y la Resolución 1715 de 2014 permite al sistema cumplir con parámetros de calidad, seguridad y eficiencia energética.
