Elementos de Control

Son los dispositivos que se encargan de ejecutar acciones para modificar el entorno del invernadero Green Life, se encuentran conectados a la Raspberry Pi mediante módulos relé, estos actúan como interruptores controlados mediante el software.

Actuadores controlados por relés:

  • 2 bombas de agua: una para riego y otra para evacuación del agua residual.
  • 2 ventiladores: ubicados en el lateral izquierdo, para disminuir la temperatura y ventilación.
  • Reflector LED: colocado en la parte superior para aumentar la temperatura y brindar luz al ají.
  • Bombillo LED (rojas y azules): Para estimular el crecimiento del ají.

Elementos de comunicación:

Estos dispositivos y tecnologías en el invernadero nos permiten obtener datos del entorno y monitorear remotamente el sistema.

  1. Sensores:
    • Sensor de temperatura DS18B20 (digital): Mide temperatura del ambiente dentro del invernadero.
    • Sensor de humedad del suelo (analógico): Mide la humedad del suelo donde se ubica el ají.
    • Sensores de nivel de agua (analógicos): En el tanque principal y la bandeja de agua residual.
  2. Cámara (Picamera2): Librería de Python que permite capturar la imagen en vivo al interior del invernadero.
  3. Raspberry PI 4 Model B:
    • Es la unidad de control y procesamiento del invernadero.
    • La Raspberry ejecuta un servidor Flask que permite recoger los datos de los sensores, controlar los relés y mostrar la interfaz web.
  4. Serveo (SSH túnel): Genera un enlace público para acceder a la interfaz web remotamente.
